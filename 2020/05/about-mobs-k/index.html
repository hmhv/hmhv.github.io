2020.05.03 В· 6 min read #iOS#Swift#MobS

MobSлһҖ?

MobSлҠ” MobX, RxSwift, CombineлЎңл¶Җн„° мқёмҠӨн”јл Ҳмқҙм…ҳмқ„ м–»мқҖгҖҒ
__SIMPLE & SAFE__мқ„ лӘ©н‘ңлЎң н•ҳлҠ” Swift лҰ¬м—‘нӢ°лёҢ лқјмқҙн”„лҹ¬лҰ¬ мһ…лӢҲлӢӨ. гҖҖ

MobSмқҳ лҢҖмғҒ мң м Җ

  • RxSwiftмқҳ к°•л Ҙн•Ём—җ мўҢм Ҳн•ң кІҪн—ҳмқҙ мһҲмңјмӢ  л¶„
  • SwiftUI+Combineл§ҢмңјлЎң к°ңл°ңмқ„ м§„н–үн•ҳкё°лҠ” м•„м§Ғ мқҙлҘҙлӢӨкі  мғқк°Ғн•ҳлҠ” л¶„

MobSмқҳ нҠ№м§•

  • к°„лӢЁн•ң мқён„°нҺҳмқҙмҠӨ
  • л©”лӘЁлҰ¬кҙҖлҰ¬лҠ” кұ°мқҳ мһҗлҸҷ
  • StateліҖкІҪм—җ мқҳн•ң Viewмқҳ к°ұмӢ мІҳлҰ¬м—җ м§‘мӨ‘
  • лӘЁл“  мІҳлҰ¬лҠ” л©”мқё мҠӨл Ҳл“ңм—җм„ң мӢӨн–ү

мҡ©м–ҙм„ӨлӘ…

Observable

ліҖмҲҳмқҳ к°’мқ„ мҶҢмң н•ҳл©° мғҲлЎңмҡҙ к°’мқҙ м„Өм •лҗ л•Ң л§ҲлӢӨ Observerм—җ нҶөм§ҖлҘј ліҙлғ„
MobX, RxSwiftмқҳ ObservableгҖҒCombineмқҳ PublisherмҷҖ лҸҷл“ұ

Observer

Observableм—җ м¶”к°Җлҗҳм–ҙ ObservableмңјлЎңл¶Җн„°мқҳ нҶөм§ҖлҘј л°ӣм•„ м•Ўм…ҳмқ„ мӢӨн–ү
MobXгҒ®action, RxSwiftгҒ®ObserverгҖҒCombineмқҳ SubscriberмҷҖ лҸҷл“ұ

Removable (Remover)

Observableм—җ м¶”к°Җлҗң ObserverлҘј ObservableлЎңл¶Җн„° мӮӯм ңн•ҳлҠ” л°©лІ•
RxSwiftмқҳ DisposableгҖҒCombineмқҳ CancellableмҷҖ лҸҷл“ұ

MobSмқҳ кё°ліё

к·ёлҹј мқјлӢЁ мҪ”л“ңлҘј нҷ•мқён•©лӢҲлӢӨ.

мқҙкІғмқҖ лІ„нҠјмқ„ лҲ„лҘјл•Ңл§ҲлӢӨ countк°Җ 1м”© мҰқк°Җн•ҳм—¬ лқјлІЁм—җ н‘ңмӢңлҗҳлҠ” мҪ”л“ңмһ…лӢҲлӢӨ.

в‘ Observableмқҳ мғқм„ұ

MobSмқҳ ObservableмқҖ property wrapperмңјлЎң кө¬нҳ„лҗҳм–ҙ мһҲм–ҙм„ң мқҙмҷҖк°ҷмқҙ @MobS.ObservableлҘј л¶ҷм—¬м„ң мғқм„ұн•©лӢҲлӢӨ.
ObservableмқҖ Observerмқҳ нҒҙлЎңм Җ м•Ҳм—җм„ң м°ёмЎ°лҗҳлҠ” мҲңк°„ мһҗлҸҷм ҒмңјлЎң ObserverлҘј мһҗмӢ м—җкІҢ м¶”к°Җн•©лӢҲлӢӨ.
кұ°кҫёлЎң Observerмқҳ нҒҙлЎңм Җ мқҙмҷём—җм„ңмқҳ м°ёмЎ°лҠ” мқјл°ҳм Ғмқё н”„лЎңнҚјнӢ°мҷҖ к°ҷмҠөлӢҲлӢӨ.

мң„мқҳ мҪ”л“ңм—җм„ңлҠ” мҙҲкё°к°’ 0мқҳ Intнҳ• ObservableгҖҒcountлҘј мғқм„ұн•©лӢҲлӢӨ.

в‘ЎObserverмқҳ мғқм„ұ & Observableм—җ м¶”к°Җ

MobSмқҳ ObserverлҠ” addObserverл©”мҶҢл“ңм—җ нҶөм§ҖмӢң мӢӨн–үн•  м•Ўм…ҳмқ„ нҒҙлЎңм ҖлЎң л„ҳкІЁм„ң мғқм„ұн•©лӢҲлӢӨ.
мғқм„ұлҗң ObserverлҠ” Observableм—җ мһҗлҸҷм ҒмңјлЎң м¶”к°Җлҗҳм–ҙ Observableм—җ мғҲлЎңмҡҙ к°’мқҙ м„Өм •лҗ  л•Ң л§ҲлӢӨ мӢӨн–үлҗ©лӢҲлӢӨ.

мң„мқҳ мҪ”л“ңм—җм„ңлҠ” countм—җ мғҲлЎңмҡҙ к°’мқҙ м„Өм •лҗ  л•Ң л§ҲлӢӨ self.countLabel.text = "\(count)"к°Җ мӢӨн–үлҗ©лӢҲлӢӨ.

addObserverлЎң м¶”к°Җлҗң ObserverлҠ” лҚ”мқҙмғҒ нҶөм§ҖлҘј л°ӣмқ„ н•„мҡ”к°Җ м—Ҷм–ҙм§Җл©ҙ мӮӯм ңн•  н•„мҡ”к°Җ мһҲмҠөлӢҲлӢӨ.
лӢӨл§Ң MobSм—җм„ңлҠ” н•„мҡ”к°Җ м—Ҷм–ҙм§„ ObserverлҠ” мһҗлҸҷм ҒмңјлЎң мӮӯм ңлҗҳлҸ„лЎқ лҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.
(мқҙкІғмқҖ RxSwiftмқҳ Disposable, DisposeBag, NSObject+Rxмқ„ н•©м№ң мІҳлҰ¬мҷҖ л№„мҠ·н•©лӢҲлӢӨ.)

мң„мқҳ мҪ”л“ңм—җм„ңлҠ” with: self нҢҢлқјлҜён„°лЎң м „лӢ¬лҗң CountUpViewControllerк°Җ deinitлҗҳлҠ” нғҖмқҙл°Қм—җ мһҗлҸҷмңјлЎң мӮӯм ңлҗ©лӢҲлӢӨ.

в‘ўObservableмқҳ к°ұмӢ 

Observableмқҳ ліҖкІҪмқҖ мқјл°ҳм Ғмқё н”„лЎңнҚјнӢ°мҷҖ к°ҷмқҙ к°’мқ„ м„Өм •н•ҳлҠ”л©ҙ лҗ©лӢҲлӢӨ. мқҙкІғмңјлЎң ObservableмқҖ мһҗмӢ м—җкІҢ м¶”к°Җлҗң лӘЁл“  Observerм—җ нҶөм§ҖлҘј ліҙлӮҙ м•Ўм…ҳмқҙ мӢӨн–үлҗ©лӢҲлӢӨ.

мң„мқҳ мҪ”л“ңм—җм„ңлҠ” count += 1лЎң Observerм—җ мһҗлҸҷмңјлЎң нҶөм§Җлҗҳм–ҙ self.countLabel.text = "\(count)"к°Җ мӢӨн–үлҗ©лӢҲлӢӨ.

л§Ҳм§Җл§үмңјлЎң

мқҙмғҒмңјлЎң MobSмқҳ кё°ліём—җ лҢҖн•ҳм—¬ м„ӨлӘ…н–ҲмҠөлӢҲлӢӨ.

мӢӨл¬ҙм—җм„ң мӮ¬мҡ©н•ҳкё° мң„н•ҙм„ңлҠ” м•„лһҳмқҳ лӮҙмҡ©лҸ„ мқҙн•ҙн•  н•„мҡ”к°Җ мһҲмҠөлӢҲлӢӨ. мһҗм„ён•ң лӮҙмҡ©мқҖ мҪ”л“ңмҷҖ мҳҲм ңлҘј нҷ•мқён•ҙмЈјм„ёмҡ”.
