2. ビルドして確認

AndroidStudioでビルドして確認

Xcodeでビルドして確認

ios/Runner.xcworkspaceを開いて

3. Today Extenstionを作成

  • テンプレートからToday Extenstion選択して作成

  • today Targetに App.framework , Flutter.framework を追加

4. Today Extenstionの修正
