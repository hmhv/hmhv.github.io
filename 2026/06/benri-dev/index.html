ブラウザだけで動く便利ツール集を作りました（AIで）

2026.06.29 · 1 min read #Web#AI#個人開発#Claude Code

AIでどこまで作れるか試していたら、便利ツール集ができていたので置いておきます。

Benri（べんり箱） — 登録不要・無料、ぜんぶブラウザの中だけで動きます。文字数カウント・QRコード・割り勘・画像リサイズなど55個。入力したデータはサーバーに送りません。

ほとんどAIに作らせた

コードはほぼ書いていません。決めたのは「何を作るか」と全体の雰囲気くらいで、あとは Claude Code に任せました。最初に何個か作って型ができると、似たツールは同じ調子でどんどん増えていきます。

少し前に「作る人」から「問う人」へという記事を書きましたが、まさにそれを地で行く感じでした。
