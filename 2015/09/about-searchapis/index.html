2015.09.15 · 7 min read #iOS#Objectivc-C#Search APIs

Search APIsとは

アプリの内容をSpotlightから検索し、検索結果からアプリに移動できる機能

Search APIsの種類

  • NSUserActivity API

iOS8で追加されたHandoff用のNSUserActivityクラスを利用する

  • CoreSpotlight API

iOS9で追加されたCoreSpotlight.frameworkを利用する

  • Web Markup API

アプリ側の対応ではなく関連ウェブページで対応
