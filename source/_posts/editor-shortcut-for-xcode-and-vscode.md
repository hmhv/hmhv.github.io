| | Xcode | VSCode |
|-|:-:|:-:|
| 行を上へ移動 | `⌘⌥[`  | `⌥↑` |
| 行を下へ移動 | `⌘⌥]`  | `⌥↓` |
| マルチカーソル | `⌃⇧↑`  | `⌘⌥↑` |
| マルチカーソル(クリック) | `⌃⇧ + Click`  | `⌥ + Click` |
| マルチカーソル(ドラッグク) | `⌥ + ドラッグ`  | `⌥⇧ + ドラッグ` |
| 次の出現箇所を選択 | `⌘⌥E` | `⌘D` |
| すべての出現箇所を選択 | (`⌘⌃E`) | `⌘⇧L` |
