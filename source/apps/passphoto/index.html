Pass Photo


  • English
  • 日本語
  • 한글
- Rich photo editing using Aviary 

- Edited photos can be stored inside the app (lockable) 

- Edited photos can be open the other app 

- If you have a copied picture, PassPhoto start in edit mode 

- If PassPhoto open by other apps with picture, PassPhoto start in edit mode

- Aviaryを利用した豊富写真編集 

- 編集後の写真はアプリ内部に保存可能 (ロック機能) 

- 編集後の写真は他のアプリで起動可能 

- 写真コピー後アプリ起動時、即編集モードで起動 

- 他のアプリで写真で起動時、即編集モードで起動

- Aviary를 이용한 막강한 사진 편집 

- 편집된 사진은 앱내부에 저장 가능(잠금 기능) 

- 편집된 사진은 다른 앱에서 열기 가능 

- 사진을 복사후 앱 기동시 즉각 편집모드로 기동 

- 단말기내의 다른 앱에서 사진 전달시 즉각 편집모드로 기동
