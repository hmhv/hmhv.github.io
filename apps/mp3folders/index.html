[MP3 Folders] is a mp3 player that plays songs from a folder (include sub folders).

** Features **
- play mp3 files from a folder (include sub folders).
- lock screen music controls
- control center music controls
- play after Interruption
- pause when earphones unplugged
- earphones remote music controls

Q : How to inport mp3 files?
A : by iTunes File Sharing. or Web browser, WebDAV client.


[MP3 Folders]はフォルダ(サブフォルダ含め)内のMP3ファイルを再生するMP3プレイヤーです。

** 機能 **
- フォルダ(サブフォルダ含め)内のMP3ファイルを再生
- ロックスクリーンからのミュージックコントロール
- コントロールセンターからのミュージックコントロール
- 割込み後の再プレイ
- イヤホンが抜けたときに再生中止
- イヤホンのリモートコントロール

Q : mp3ファイルをインポートする方法は？
A : iTunesのファイル共有を利用します。又はウェブブラウザ、 WebDAVクライアントでも可能です。


[MP3 Folders]는 폴더(서브폴더 포함)내의 MP3파일을 재생하는 MP3플레이어 입니다.

** 기능 **
- 폴더(서브폴더 포함)내의 MP3파일을 재생
- 락스크린에서 음악 컨트롤
- 컨트롤 센터에서 음악 컨트롤
- 중단후 다시 재생
- 이어폰 뽑을시 재생 정지
- 이어폰에서 음악 컨트롤

Q : mp3파일을 임포트하는 방법은?
A : iTunes의 파일 공유 기능을 이용합니다. 또는 웹 브라우져, WebDAV클라이언트로도 가능합니다.
